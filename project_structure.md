# 📁 DC-Detector Project Structure

Новая архитектура проекта для лучшей организации кода.

## 🏗️ Структура проекта

```
DC-Detector/
├── 📁 src/                          # Исходный код
│   ├── 📁 app/                     # Основное приложение
│   │   ├── app.py                  # Flask приложение (стандартная версия)
│   │   ├── app_pi.py               # Flask приложение для Raspberry Pi
│   │   └── app_websocket.py        # WebSocket версия
│   ├── 📁 config/                  # Конфигурация
│   │   ├── config_pi.py           # Конфигурация для Raspberry Pi
│   │   └── config_default.py      # Стандартная конфигурация
│   └── 📁 utils/                   # Утилиты
│       ├── camera_utils.py         # Утилиты для работы с камерой
│       ├── detection_utils.py      # Утилиты для детекции
│       └── system_utils.py         # Системные утилиты
├── 📁 scripts/                     # Скрипты установки и управления
│   ├── setup.sh                   # Основной скрипт установки
│   ├── fix_github_cli.sh          # Исправление GitHub CLI
│   ├── quick_start.sh             # Быстрый запуск
│   ├── check_system.sh            # Диагностика системы
│   ├── read_logs.sh              # Анализ логов
│   └── commit_logs.sh            # Коммит логов
├── 📁 logs/                        # Логи
│   ├── setup_log_*.log           # Логи установки
│   ├── app_log_*.log             # Логи приложения
│   └── error_log_*.log           # Логи ошибок
├── 📁 models/                      # Модели машинного обучения
│   ├── bestfire.pt               # Модель детекции огня
│   └── yolov8n.pt                # Базовая YOLO модель
├── 📁 static/                      # Статические файлы
│   ├── 📁 css/                    # Стили
│   │   └── styles.css
│   └── 📁 js/                     # JavaScript
│       ├── script.js
│       └── opencv-fire-detector.js
├── 📁 templates/                   # HTML шаблоны
│   └── index.html
├── 📁 docs/                        # Документация
│   ├── README.md                  # Основная документация
│   ├── README_LOGS.md            # Документация по логам
│   ├── INSTALL.md                # Инструкции по установке
│   └── TROUBLESHOOTING.md        # Устранение неполадок
├── 📁 tests/                       # Тесты
│   ├── test_detection.py
│   ├── test_camera.py
│   └── test_system.py
├── 📁 examples/                    # Примеры использования
│   ├── basic_usage.py
│   └── advanced_usage.py
├── 📁 deployment/                  # Развертывание
│   ├── 📁 docker/                 # Docker конфигурация
│   │   ├── Dockerfile
│   │   └── docker-compose.yml
│   └── 📁 systemd/                # Systemd сервисы
│       └── fire-detection.service
├── requirements.txt               # Python зависимости
├── .gitignore                    # Git исключения
└── setup.py                      # Установка Python пакета
```

## 🎯 Преимущества новой структуры

### ✅ Организация:
- **Четкое разделение** по функциональности
- **Логическая группировка** файлов
- **Легкая навигация** по проекту

### ✅ Масштабируемость:
- **Модульная архитектура** - легко добавлять новые компоненты
- **Разделение ответственности** - каждый модуль отвечает за свою область
- **Переиспользование кода** - общие утилиты в отдельной папке

### ✅ Поддержка:
- **Централизованная документация** в папке docs/
- **Структурированные логи** в папке logs/
- **Отдельные тесты** для каждого компонента

### ✅ Развертывание:
- **Docker поддержка** для контейнеризации
- **Systemd сервисы** для автозапуска
- **Гибкая конфигурация** для разных окружений
